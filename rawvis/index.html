Progressive & Adaptive Processing for
In situ Big Raw Data Visualization & Analytics
>_ the framework

RawVis framework enables in site query processing on big raw data files for interactive visual exploration scenarios, using commodity hardware.
>_ in situ processing

RawVis built on top of a lightweight in-memory index, that is constructed on-the-fly given the first user query over a raw file and adapted incrementally based on the user interaction.
