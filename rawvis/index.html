Progressive & Adaptive Processing for
In situ Big Raw Data Visualization & Analytics
>_ the framework

RawVis framework enables in-situ query processing on big raw data files for interactive visual exploration and analytics, using commodity hardware.
>_ progressive & adaptive indexing

RawVis built on top of a in-memory index, that is constructed on-the-fly given the first user interaction, and progressively adapt its structure based on user interactions.
